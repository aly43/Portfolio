What is Twine?

Twine is a platform that allows users to create interative stories based on user input. Sort of like a interactive adventure game. My project on twine uses the twine engine to create an interactive story about burnout. I thought it was relevent, given the world was still recovering from the pandemic. I heard a lot about the term burnout, and even experienced it myself with academics. The story follows someone with a corporate worklife, and has multiple endings based on user input. The experience is to draw attention to the causes and symptoms of burnout.
